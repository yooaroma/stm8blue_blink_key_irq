## GPIO 설명

- PB5 : LED
- PD3 : KEY
- PD5 : UART1_TX
- PD6 : UART1_RX
- PC7 : SPI_MISO
- PC6 : SPI_MOSI
- PC5 : SPI_SCK
- PA3 : SPI_NSS
- PD2 : AIN3
- PB5 : I2C_SDA
- PB4 : I2C_SCL
